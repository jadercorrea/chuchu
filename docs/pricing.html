Frequently Asked Questions

Do I need to pay for the CLI?
No. The CLI is completely free and open-source. You only pay for your own API keys (like Groq at $0-5/month or free with Ollama locally).
What's the difference between CLI and Live?
The CLI is the core tool that runs locally on your machine. Live is a web dashboard that adds real-time monitoring, session history, team collaboration, and analytics on top of your CLI usage.
Can I use GPTCode with my company's API keys?
Yes! GPTCode is designed for BYOK (Bring Your Own Keys). Configure your OpenAI, Anthropic, Groq, or any OpenAI-compatible endpoint. Your keys stay on your machine.
Is there a free trial for Live Pro?
Yes, Live Pro includes a 14-day free trial with full features. No credit card required to start.
How is this cheaper than Cursor or Copilot?
GPTCode lets you use any LLM provider, including free ones like Ollama or ultra-cheap ones like Groq (~$0.05/100K tokens). You pay the actual API cost, not a markup. Most developers spend $2-5/month on API calls.
What if I want to self-host Live?
Contact us for enterprise self-hosted licensing. The Live dashboard is built with Phoenix LiveView and can run on your infrastructure.
